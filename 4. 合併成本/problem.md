## Task Description

有
n
個數字排成一列，依序是
a[1],a[2],a[3],⋯,a[n]
。

每次可以挑選兩個相鄰的數字
(u,v)
合併，合併會花費
|u−v|
的元，合併起來的數字會變為
u+v
。

問把
n
個東西合成一個數字的最小花費是多少？
