## Task Description

給一個二維平面，座標如同數學的二維座標(Ｙ正為北，Ｘ正為東)。起始位置在 (0, 0)，接下來會有
n
個座標，你需要按照這些座標點的順序移動，保證僅會垂直或水平方向上移動，不會斜向移動，且第一個點保證一定是Ｘ軸正的位置(初始方向向右)。

請輸出這條路徑中，左轉、右轉、迴轉的個數分別為多少。

子問題一 (60%)
n=2

子問題二 (40%)
n≤100
