## Task Description

在
n
個人之中，每個人被分配到一個號碼，記錄為
a1,a2,…,an
，這些號碼彼此都不相同，我們要找出當中最幸運的一個
已知在
[L,R]
區間之中，最幸運號碼定義為：

* 當
L=R
時，幸運號碼為
a[L]
* 當
L<R
時，找出當中最小號碼的位置
m
，並以最小號碼所在的位置將整個區間區分為左邊
[L,m−1]
與右邊
[m+1,R]
，計算兩個區間各自的總和，總和比較大的區間即為幸運號碼所在的區間；如果兩個區間的總和相等，則幸運號碼位在右邊的區間，若區間不包含任何數字，總和視為
0


重複以上步驟直到收斂至1. 時即可找到幸運號碼
請找出這
n
個人之中的幸運號碼
