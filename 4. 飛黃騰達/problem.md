## Task Description

飛黃是一種生物，活在二維座標平面上。

有隻特別的飛黃一開始在座標 (0, 0) 的位置，而且你知道它只會往右上方移動，也就是移動的時只可以走到 x 座標跟 y 座標都不比原本小的位置。

現在座標平面的第一象限上有
n
個位置有果實，給定這
n
個果實的座標，你想要知道這隻特別的飛黃最多可以吃到幾個果實（它必須移動到果實所在的座標才可以吃到果實）。
