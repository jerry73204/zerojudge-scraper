## Task Description

有一根長度為
L
的棍子，你會把這個棍子切割
n
次。

假設一開始棍子左端放在數線上
0
的位置，棍子的右端放在數線上
L
的位置，每次的切割會給定一個介於
0
到
L
的數字表示要切個的位置，你要把穿過個這位置的棍子切成兩段，而所需的花費就等於所切割的棍子的長度。
