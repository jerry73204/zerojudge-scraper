## Task Description

給一個長度為
n
的正整數序列
a1,a2,…,an
，你可以執行多次操作 (包含 0 次)，每次操作只能選擇這個序列的第一個或最後一個數字，再將這個數字從序列中刪除並自己搜集起來。

求滿足總和不超過
k
且搜集的數字奇數和偶數個數相同的條件下，所能搜集的數字總和最大為多少。
