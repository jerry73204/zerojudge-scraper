## Task Description



一個雷射光從 (0,0) 向右邊發射，平面上有很多個鏡子，問雷射光會反射幾次，保證輸入沒有無限循環的情形。

鏡子用三個數字表示
(xi,yi,ti)
，代表座標在
(xi,yi)
上。
ti=0
代表示這種 / 擺放方式的鏡子，
ti=1
代表這種 \ 擺放方式的鏡子，保證不會有一個位置有多個鏡子。
