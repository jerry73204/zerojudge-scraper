## Task Description

輸入一個長度為
2n
的陣列，其中
1
~
n
的每個數字都剛好各 2 次。

i
的低窪值的定義是兩個數值為 i 的位置中間，有幾個小於 i 的的數字。

以 $[3, 1, 2, 1, 3, 2] 為例，1的低窪值為 0, 2 的低窪值值為 1, 3 的低窪值為 3。

請對於每個
1
~
n
的數字都求其低窪值（兩個相同的數字之間有幾個數字比它小），輸出低窪值的總和，答案可能會超過 C++ int 的上限。
