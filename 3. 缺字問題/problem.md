## Task Description

給定一個大小為
K
個字母的集合和字串
S
，求出在使用該集合所組出長度為
L
字串中，不為字串
S
子字串的最小字典序字串為何。

例如字母集合
{a,c,m}
，其能組出長度為
2
的字串字典序由小到大排序為
aa<ac<am<ca<cc<cm<ma<mc<mm
。字串
S
為
accaamcm
，因此
ma
為不在
S
內的最小字典序字串。
